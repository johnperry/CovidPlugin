Open Quantitative Imaging for COVID-19 (Open-QIC) Repository

Open-QIC is a new open access CT image data and clinical metadata repository created to help accelerate Artificial Intelligence research and clinical research into COVID-19.


Clinical sites can use the open source COVIDClient java application to de-identify and upload longitudinal CT imaging and clinical metadata from COVID-19 patients to the Open-QIC repository. 		COVIDClient Java Application

Open-QIC provides a listing of publicly available data for individual or full collection download. Links are also provided to view metadata and CT image slices. 		Open-QIC Data Repository
